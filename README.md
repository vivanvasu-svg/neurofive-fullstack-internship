# NeuroFive Solutions – Full Stack Developer Internship

This repository contains the tasks completed as part of the **Full Stack Developer Internship** assessment at **NeuroFive Solutions**. The project is built with **React + Vite** and focuses on responsive frontend development and API integration.

## Task 1 – Responsive Landing Page

Built **Brew Haven**, a modern coffee shop landing page featuring:

- Responsive design (mobile, tablet, and desktop)
- Reusable React components
- Clean and consistent UI
- Modern layout with CSS Modules

## Task 2 – Public API Integration

Extended the project by integrating **TheCocktailDB API**.

Features include:

- Fetching live drink data
- Search functionality
- Loading and error states
- Responsive card layout

## Tech Stack

- React
- Vite
- CSS Modules
- React Icons
- TheCocktailDB API

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
